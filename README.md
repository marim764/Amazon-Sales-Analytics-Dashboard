# Amazon Sales Analytics Dashboard
## 📌 Project Overview
An interactive Power BI dashboard developed as a team project during the **Digital Egypt Pioneers Initiative (DEPI)**. The project analyzes Amazon product performance over a one-month period to provide actionable intelligence on sales velocity and revenue drivers
## ⚙️ Workflow
1. **Cleaning:** Handled missing values and standardized data formats in Excel.
2. **Modeling:** Built a **Star Schema** to connect Fact sales with Dimensions (Categories, Dates).
3. **Visualization:** Created 3 main pages: Overview, Drivers, and Product Deep Dive.
